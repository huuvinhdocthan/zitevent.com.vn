# ZIT Event Static Website

Website tĩnh chuyên nghiệp cho công ty ZIT Event, sử dụng HTML5, CSS3, JavaScript và Bootstrap 5.

## Cấu trúc

- `index.html` - Trang chủ
- `about.html` - Giới thiệu
- `services.html` - Dịch vụ
- `equipment.html` - Thiết bị cho thuê
- `pricing.html` - Bảng giá
- `projects.html` - Dự án tiêu biểu
- `news.html` - Tin tức
- `contact.html` - Liên hệ
- `assets/css/styles.css` - Tùy chỉnh giao diện
- `assets/js/main.js` - Tương tác và lọc dữ liệu

## Tính năng

- Responsive trên desktop và mobile
- Header cố định và nút quay lên đầu trang
- Tìm kiếm và lọc thiết bị bằng JavaScript
- Sắp xếp bảng giá theo tên hoặc giá
- Form liên hệ kiểm tra dữ liệu phía client
- Tối ưu cho SEO cơ bản và hiển thị trên GitHub Pages / Netlify / Vercel

## Triển khai

Chỉ cần tải toàn bộ thư mục lên hosting tĩnh bất kỳ.
